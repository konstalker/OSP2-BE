Автоматическая загрузка последней версии OSP2-BE
Как пользоваться: 
1. Поместить файл Updater OSP2-BE для вашей операционной системы в папку Quake 3
2. Запустить
3. Дождаться окончания загрузки
4. Нажать любую кнопку
